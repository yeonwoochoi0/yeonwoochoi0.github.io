## Git
- 분산관리 시스템
- 빠른 속도
- 오픈소스 형식
- [repository] : https://git.kernel.org/pub/scm/git/git.git

## Git 명령어
- git init : 현재 작업중인 디렉토리를 git 저장소로 지정
- git status : 현재 Git 상태 확인
- git commit : 새로운 commit 생성
- git log : commit 기록 확인

### Git branch
